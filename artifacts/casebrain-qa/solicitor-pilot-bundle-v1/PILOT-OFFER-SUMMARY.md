# Pilot offer summary

**Type:** Controlled **30-day supervised** pilot — not a firm-wide rollout  
**Proof baseline:** Controlled fictional + gold pack PASS · not solicitor-validated on live files yet  
**Full offer (repo / on request):** `docs/pilot-trust-pack/CONTROLLED-PILOT-OFFER.md` · short form is this file

---

## In one paragraph

Limited named users review gold packets (Waves A+B / full 20) and, only after kick-off, a small number of **firm-redacted** matters. Outputs are drafts. Solicitor sign-off before any external use. No autonomous sending. Success = gaps spotted, unsafe assumptions blocked, usable chase/court after edit, clearer client summaries where used, time saved, over-warning tracked — not “AI accuracy guaranteed”.

---

## Limits

| Limit | Rule |
|-------|------|
| Duration | 30 days (extend only in writing) |
| Users | Named; typically 1–5 |
| Matters | Written cap |
| Material | Gold first; then redacted shadows |
| Sending | None autonomous |
| Responsibility | Remains with the firm / instructing solicitor |

---

## Success criteria (short)

- Material disclosure gaps worth putting on the record  
- Served / referred / missing / unsafe-to-say handled honestly  
- CPS chase: accept or light edit  
- Court notes usable after review  
- Client summaries provisional and clear (where used)  
- Net time saved on first-pass triage  
- False positives / over-warning tracked  
- **Hard safety failures: 0**

Detail (repo / on request): `docs/pilot-trust-pack/PILOT-SUCCESS-CRITERIA.md`

---

## Kick-off before firm uploads

1. Named users + matter cap  
2. Claim discipline acknowledged  
3. Subprocessors + no-training terms confirmed  
4. Deletion / incident contacts filled  
5. Start gold Waves A+B (this bundle)
