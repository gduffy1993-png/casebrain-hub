# CaseBrain Roadmap: Stage 1 to Launch

## Stage 1: Engine Lock (Current)

Goal: finalize deterministic Q1-Q10 quality and freeze core behavior.

1. Run a 20-case confidence check focused on Q1 + Q7.
2. Verify:
   - no duplicate-root picks in single answers,
   - no repetitive reliability spam,
   - reasons match type (contradiction vs chronology/continuity vs reliability).
3. If stable, freeze engine changes to hotfix-only.

Exit criteria:
- Consistent passes across repeated runs,
- no regressions in Q6 and Q9 stability.

---

## Stage 2: CPS / Prosecution Pressure Layer

Goal: move from analysis to adversarial strategy.

1. Expand CPS pressure lens:
   - Crown likely repair targets,
   - defence pre-emption actions this week.
2. Add counter-move logic:
   - "if Crown fixes X, defence should do Y."
3. Keep additive and isolated (do not destabilize Q1-Q10 path).

Exit criteria:
- Evidence-grounded outputs,
- deterministic behavior,
- clear tactical value for timing decisions.

---

## Stage 3: Chat Orchestration Layer

Goal: make the system naturally usable in live solicitor workflow.

1. Intent router for natural asks (e.g., "can I win this?", "what do I do this week?").
2. Module planner to run the right internals (Q1/Q2/Q3/Q6/Q7/Q9 + CPS lens).
3. Response composer that returns:
   - current position,
   - what can swing the case,
   - immediate action plan.

Exit criteria:
- Conversational usability without losing grounding,
- consistent answer quality across varied prompts.

---

## Safety and Quality Guardrails (Always On)

1. No invented evidence, no invented exhibits.
2. Use "not stated in the materials" where evidence is missing.
3. Keep court-safe wording (no unsupported speculation).
4. Maintain deterministic guardrails for critical outputs.

---

## Validation Sequence

1. Stage 1 final confidence pass (20 cases, Q1 + Q7 deep check).
2. Stage 2 tactical value pass (CPS lens usefulness).
3. Stage 3 real-world prompt pass (solicitor-style questions).
4. Demo readiness check:
   - upload -> strategy output,
   - natural Q&A,
   - grounded tactical actions.

---

## Positioning

Current trajectory:
- from "useful legal analysis tool"
- to "must-have criminal defence strategy copilot."
