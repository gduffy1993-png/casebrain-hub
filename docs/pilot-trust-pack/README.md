## What this pack is

A short, solicitor-facing set of documents for a **supervised 30-day controlled pilot**.

It explains:

- what CaseBrain has already proved on controlled fictional / PDF-backed bundles
- how a firm can trial the product safely (Waves A+B gold review, or redacted shadow matters)
- claim discipline, data handling, subprocessors, deletion, and success criteria

This pack is **docs only**. It does not change product code.
