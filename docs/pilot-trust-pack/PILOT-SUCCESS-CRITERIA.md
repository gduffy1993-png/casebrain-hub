# Pilot success criteria

**Pilot type:** 30-day supervised controlled pilot  
**Proof already in hand:** controlled fictional scale + gold manual PASS (hard safety clean)  
**This document measures:** whether the pilot creates **solicitor trust and practical value** — not whether marketing claims can expand

---

## Primary success signals

| Signal | What “good” looks like |
|--------|------------------------|
| Gaps spotted | Reviewers find material served / referred / missing issues they would want on the record |
| Unsafe assumptions blocked | Tool caveats or blocks “treated as served” / overstatement where papers do not support it |
| Chase drafts | Accepted as-is or lightly edited; not discarded as generic noise |
| Court notes | Usable after solicitor review; family-specific; no plea/outcome advice |
| Client summaries | Clearer than raw papers; provisional; not advice |
| Time saved | Net time saved on first-pass disclosure triage on sample matters |
| Source confidence | Reviewers understand anchors / “source verification required” as honesty, not fake certainty |
| False positives / over-warning | Tracked; rate acceptable to reviewers (noise does not drown signal) |

---

## Scoring the pilot (end of 30 days)

Collect completed `SOLICITOR-REVIEW-FORM.md` sheets.

| Outcome | Rule of thumb |
|---------|----------------|
| **Pilot success** | Majority PASS; WARNs are presentation-only; zero unresolved hard-safety FAILs; firm willing to continue or expand carefully |
| **Partial success** | Value on gold + some shadows, but wording/noise needs another polish cycle before wider use |
| **Not ready to expand** | Hard safety FAIL, systematic false served, or firm would not trust outputs even with sign-off |

---

## Metrics to record (simple)

- Number of matters reviewed (gold vs redacted)  
- PASS / WARN / FAIL counts  
- Hard safety incidents (target: **0**)  
- Chase: % accept / light edit / rewrite / discard  
- Court: % usable after review  
- Estimated minutes saved (or lost) per matter  
- Top 5 recurring WARN themes  
- Deletion requests completed  

Do not invent precision. Small-n solicitor judgment beats fake dashboards.

---

## Explicit non-goals for this pilot

- Proving nationwide real-world accuracy  
- Replacing solicitor review  
- Enabling autonomous sending  
- Claiming SOC 2 / ISO / SRA endorsement from pilot participation  

---

## Link to next gate

If success: agree either (a) extended supervised pilot, or (b) limited paid use under the same claim discipline.  
If partial: fix presentation / process, re-run gold + one shadow.  
If not ready: stop expansion; preserve evidence; fix hard issues first.
