# Claim discipline

**Rule:** Prefer under-claiming. Solicitors trust caution more than hype.

---

## Can claim

| Claim | Notes |
|-------|-------|
| Controlled fictional / PDF-backed proof at scale | ~3,000 messy bundles; hard safety clean in that programme |
| Targeted regression lock | 140-case lock; hard safety clean |
| Gold Manual Proof Set v1 | 20 pass / 0 warn / 0 fail; hard safety 0 |
| Waves A+B reviewer-style PASS | Supervised review aid on controlled packets |
| Full 20 presentation PASS after polish | Still not solicitor-signed real-world validation |
| Source-linked / evidence-state outputs | Served / referred / missing / unsafe-to-say style discipline |
| Supervised review aid for disclosure traps | Human must sign off before external use |

---

## Cannot claim

| Claim | Why not |
|-------|---------|
| Solicitor-validated real-world performance | Checklists / live shadow pilot not completed as industry proof |
| Legal advice replacement | Outputs are drafts; solicitor remains responsible |
| Autonomous court / CPS / client sending | Explicitly out of scope |
| Guaranteed accuracy on every file | No product can honestly claim this |
| SOC 2 / ISO 27001 / pen-test / SRA approval | Not asserted in this pack |
| “AI proves guilt / innocence” | Hard-safety forbidden territory |
| Ready for unsupervised firm-wide rollout | Pilot is supervised and limited |

---

## Wording — website

**Safe:**

> CaseBrain helps criminal defence teams spot disclosure traps — what is served, referred only, missing, or unsafe to say — with source-linked review aids. Controlled fictional proof is complete; supervised pilot next. Not legal advice.

**Avoid:**

> Guaranteed court-ready. Replaces your solicitor review. Autonomously chases CPS. Certified / SRA-approved AI.

---

## Wording — demo

**Safe:**

> This is a controlled / fictional (or redacted pilot) matter. Treat every line as provisional until you check the papers. CaseBrain is a review aid, not a substitute for your judgment.

**Avoid:**

> The system has proved the case. You can send this as-is.

---

## Wording — pilot invite

**Safe:**

> We have finished controlled fictional proof (scale + gold manual pack, hard safety clean). We are inviting a small number of firms to a supervised 30-day pilot — gold packets and/or redacted bundles — with solicitor sign-off on anything leaving the firm.

**Avoid:**

> We are live-validated on real client outcomes nationwide.

---

## Feedback vs training

Anonymised solicitor **form themes** (what worked / what did not) may inform product process.  
That is **not** permission to train public models on firm uploads or redacted bundle text.

---

## Internal reminder

If a sentence would still be true after removing “AI”, keep it.  
If it only works as marketing magic, cut it.
