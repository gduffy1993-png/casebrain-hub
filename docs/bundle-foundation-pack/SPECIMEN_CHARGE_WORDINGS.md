## Particulars template rules
- Use `Statement of Offence` and `Particulars of Offence` for indictment-style pages.
- Use one count per offence unless the recipe deliberately tests multiple counts.
- Particulars must include date or date range, place, complainant/property/drug/vehicle detail as applicable.
- Do not add aggravating facts unless the MG5/MG11/exhibit material contains them.
- Mark uncertain statutory wording as `verify against [URL]` in the recipe notes.
