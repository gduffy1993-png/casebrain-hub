# PILOT_READINESS.md

## Why the foundation matters

The chaos factory has already tested safety against bad paper. Paid pilot trust needs the opposite: clean ordinary papers that let a solicitor see value quickly.

A pilot solicitor will judge CaseBrain in roughly 30 seconds:

1. Court Today shows the right client, court and time.
2. Case header is one clean row.
3. Control Room gives a short useful summary.
4. Disclosure Chase gives practical missing-item bullets.
5. War Room wording is safe and conditional where needed.
6. No raw glue, tracker text or demo wording appears.

## Foundation pack role

The foundation pack supplies normal bundle shapes:

- S recipes: clean small magistrates files.
- M recipes: normal messy files where amber is expected.
- B recipes: Crown/PTPH files where header must be solid and strategy can be provisional.

## Pilot bar

For the first pilot set, generate only S1-S2 samples first.

Acceptance:

- boring PDF
- clear header fields
- tracker matches PDF
- upload works first time
- Court Today correct
- Disclosure Chase short and practical
- no over-amber language where the file is clean

## Real anonymised matters later

Eventually, trust improves with 5-10 properly redacted real firm matters reviewed with consent. That is separate from this synthetic/public-source foundation. Do not use real matter material in this pack.

## Not a chaos replacement

Do not continue CB-TB chaos batches from this folder. Chaos and foundation are separate lanes.
