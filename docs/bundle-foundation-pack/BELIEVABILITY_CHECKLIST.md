# BELIEVABILITY_CHECKLIST.md

## Obvious / bad

1. Giant `FICTIONAL TEST DATA ONLY` banner on every page.
2. AI-style narrative headings instead of MG/form headings.
3. Every pack starts with identical page order and identical wording.
4. All cases have dramatic missing evidence traps.
5. Real court names mixed with fake defendants in a way that suggests live listing data.
6. Unrealistic URNs such as TEST-123 repeated.
7. MG11 witness statement written like a blog post.
8. Charge line says only `theft case` without statute.
9. All evidence appears fully served and perfect.
10. PDF has tracker truth keys printed inside it.
11. Court Today shows raw `CourtHearing` glue.
12. Defendant name has DOB stuck to surname.
13. Witness `Statement` tail included in complainant name.
14. All packs have exactly the same number of pages.
15. Crown indictment appears in tiny SJP case with no reason.
16. MG6 schedule lists impossible items unrelated to offence.
17. Medical evidence is asserted but no note/report exists.
18. CCTV is described as conclusive without being served.
19. Random form titles invented by AI.
20. Long paragraphs explain the app rather than being case papers.

## Believable / good

1. No big fiction banner; use fictional data silently in ordinary form style.
2. Use real-style MG titles and public form labels.
3. Plausible fake URNs with varied but consistent pattern.
4. Clean mags packs are short, ordinary and extractable.
5. Messy packs have one or two normal tensions, not five chaos traps.
6. Crown packs have proper index and page ranges.
7. Charge wording includes statute line.
8. Listing notice has a clear court/date/time line.
9. MG5 is short, balanced and chronological.
10. MG11 reads like a witness/officer account, not an essay.
11. MG6C is table-like with item/location/decision columns.
12. Served vs missing material is visible in file shape.
13. Co-defendant names appear only where recipe needs them.
14. One rough scan page only in messy packs, not clean pilot packs.
15. Header fields are boring and obvious.
16. Page counts vary by recipe.
17. SJP pack is thin because SJP is thin, not because material is broken.
18. Disclosure chase follows the actual schedule gaps.
19. Tracker truth stays outside the PDF.
20. Court Today can show the right case in under 30 seconds.

## Final visual test
- Open the PDF without CaseBrain.
- Ask: would a duty solicitor recognise this as a normal file shape?
- If it looks like AI training material, reject it before factory run.
