# Pre-hearing Readiness Badge (slice 1)

**Status:** Local computed badge — not case strength, not plea advice.

**Flags:** `?reasoningV2=1` **and** `?readiness=1` (both required). `localStorage: casebrain:readiness=true`.

**Not stored:** readiness output, bundle text, evidence, paths.

**Inputs:** Reasoning V2 view model; optional Client Stress result from saved selection; bundle/hearing metadata from Control Room.
